# awesome-ai-native-interview

一个面向 **AI Native 工程师** 的面试训练与内容沉淀仓库。

每天一道题，先作答，再复盘，把真正有价值的内容沉淀下来。  
前期以前端方向为主，后续逐步扩展到客户端、服务端和 AI 工程。
