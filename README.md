# awesome-ai-native-interview

AI Native 工程师面试题库：围绕 Context Engineering、RAG、Tool Use、Agent Workflow、Evaluation、Observability 与 AI Product Design，沉淀每日面试训练题与核心概念。

## 在线站点

本仓库使用 [Rspress](https://rspress.rs/zh/guide/start/introduction) 编译静态文档站点，站点首页内容维护在 [`index.md`](index.md)。

- Cloudflare Pages: <https://awesome-ai-native-interview.pages.dev/>
- 面试题索引与核心概念：[`index.md`](index.md)
- 每日训练记录：[`daily/`](daily/)

## 本地静态站点预览

```bash
npm install
npm run dev
```

静态资源编译：

```bash
npm run build
```

构建产物输出到 `doc_build/`。
